🔥 Firebase 연동 설정
⚠ Firebase 미연결 — 로컬 데이터만 사용 중 (새로고침 시 초기화)
1
Firebase 프로젝트 생성
1. console.firebase.google.com 접속 2. [프로젝트 추가] → 이름: rental-manager 3. Google Analytics 비활성화 (선택) 4. [프로젝트 만들기]
2
Realtime Database 활성화
1. 좌측 메뉴 → [빌드] → [Realtime Database] 2. [데이터베이스 만들기] → 지역: asia-southeast1 3. 보안 규칙: [테스트 모드] 선택 후 시작 (30일 후 아래 규칙으로 교체 필요) 보안 규칙 예시: { "rules": { ".read": "auth != null", ".write": "auth != null" } }
3
웹 앱 등록 → Firebase Config 복사
1. 프로젝트 개요 → [앱 추가] → [웹(</>)] 2. 앱 닉네임: rental-web 3. [앱 등록] → firebaseConfig 복사 예시: const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "rental-xxx.firebaseapp.com", databaseURL: "https://rental-xxx-default-rtdb.asia-southeast1...", projectId: "rental-xxx", storageBucket: "rental-xxx.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc..." };
4
Firebase Config 입력 (복사한 값 붙여넣기)
주의: Database URL이 가장 중요합니다. 반드시 https://로 시작하는 전체 URL을 입력하세요.
📥 CSV 파일로 데이터 가져오기
엑셀 또는 구글시트 → CSV로 저장 → 아래에 업로드하면 앱과 Firebase에 자동 반영됩니다.
파일 3개 (임차인 / 청구 / 입금) 를 각각 업로드하세요. 순서 무관합니다.

청구ID,임차인,층,청구월,유형,월세,부가세,관리비,합계,입금액,미납금,상태,납부기한,비고
입금 CSV
입금ID,임차인,청구ID,입금일,입금액,비고

📊 Google Sheets 연동
⚠ Google Sheets 미연결 — Apps Script URL 미설정
1
Google Sheets 준비
1. sheets.google.com → 새 스프레드시트 생성 2. 파일명: 임대료관리_동기화 3. 하단 시트 탭 3개 생성: - Tenants (임차인 정보) - Bills (청구 내역) - Payments (입금 내역) 4. 확장 프로그램 → Apps Script 클릭
2
Apps Script 코드 (전체 복사 후 붙여넣기)
function doPost(e) { try { const data = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.getActiveSpreadsheet(); if (data.action === 'export') { const sheet = ss.getSheetByName(data.sheet); if (!sheet) return res('Sheet not found: ' + data.sheet); sheet.clearContents(); if (data.rows && data.rows.length > 0) { sheet.getRange(1,1,data.rows.length,data.rows[0].length) .setValues(data.rows); sheet.getRange(1,1,1,data.rows[0].length) .setFontWeight('bold').setBackground('#E8F0FE'); } return res('OK'); } if (data.action === 'import') { const result = {}; ['Tenants','Bills','Payments'].forEach(name => { const sheet = ss.getSheetByName(name); result[name] = sheet ? sheet.getDataRange().getValues() : []; }); return res(JSON.stringify(result)); } } catch(err) { return res('Error: ' + err.message); } } function doGet(e) { return res('Rental Sync API v2'); } function res(msg) { return ContentService.createTextOutput(msg) .setMimeType(ContentService.MimeType.TEXT); }
3
웹앱으로 배포
1. [배포] → [새 배포] → 유형: 웹 앱 2. 다음 사용자로 실행: 나 3. 액세스 권한: 누구나 (익명 포함) 4. [배포] → 권한 승인 → URL 복사
4
웹앱 URL 입력
5
자동 내보내기 설정
이 설정은 앱 → 구글시트 내보내기용입니다.
구글시트 데이터를 앱으로 가져오려면 아래 [📥 시트에서 가져오기] 버튼을 사용하세요.
